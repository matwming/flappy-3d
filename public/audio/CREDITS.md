1. Visit https://pixabay.com/sound-effects/ (CC0, no attribution required)
2. Search terms: "whoosh short" (flap), "ding bright" (score), "thud cartoon" (death), "ambient game loop" (music)
3. Download and replace the placeholder files in `public/audio/`
4. Update this CREDITS.md with the actual source URLs
