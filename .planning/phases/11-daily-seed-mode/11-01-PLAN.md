# Phase 11 Plan 01: Daily-Seed Mode

## Objective

Add deterministic daily obstacle layouts (mulberry32 seeded RNG), daily attempt tracking in StorageManager, "Today's best" UI on Title, and a Share button on GameOver for daily mode.

## Context

- Phase 9 added mode infrastructure (GameMode type, SET_MODE, StorageManager v3 with dailyAttempts placeholder)
- Phase 10 added TimerSystem for time-attack; pattern for mode-conditional game-loop behavior established
- ObstacleSpawner.step uses Math.random() for gapCenterY — single swap point

## Tasks

### Task 1: Seeded RNG + ObstacleSpawner integration (MODE-07)

**type:** auto

- Create `src/utils/rng.ts`: mulberry32(seed) → () => number, dailySeed() → number, todayDate() → string
- Modify `src/systems/ObstacleSpawner.ts`: add private rng field (default Math.random), add setRng() method, replace Math.random() call in step() with this.rng()
- Modify `src/main.ts` roundStarted handler: after existing resets, set spawner.setRng(mulberry32(dailySeed())) when mode === 'daily', else spawner.setRng(Math.random)

**Verify:** tsc --noEmit exits 0; grep confirms mulberry32 in rng.ts + used in main.ts

### Task 2: Daily attempt tracking (MODE-08)

**type:** auto

- Modify `src/storage/StorageManager.ts`: add getDailyAttempt(date) + recordDailyAttempt(date, score) methods
- Modify `src/main.ts`: in actor.subscribe gameOver branch, after pushLeaderboard, call storage.recordDailyAttempt(todayDate(), score) when mode === 'daily'
- Modify `src/ui/screens/TitleScreen.tsx`: add storage prop; when mode === 'daily', render .daily-stats paragraph with today's best / first attempt text
- Add `.daily-stats` CSS rule to `src/ui/styles.css`
- Update UIBridge.tsx to pass storage to TitleScreen

**Verify:** tsc --noEmit exits 0; grep confirms getDailyAttempt|recordDailyAttempt in StorageManager + TitleScreen + main.ts

### Task 3: Share button on GameOver (MODE-09)

**type:** auto

- Modify `src/ui/screens/GameOverScreen.tsx`: add mode prop; when mode === 'daily', render Share button with useState for "Copied!" feedback
- Modify `src/ui/UIBridge.tsx`: pass mode prop to GameOverScreen (via snap.context.mode)

**Verify:** tsc --noEmit exits 0; grep confirms clipboard.writeText|Share in GameOverScreen.tsx

## Success Criteria

- mulberry32 defined in rng.ts, used in main.ts
- dailySeed + todayDate in rng.ts
- setRng defined in ObstacleSpawner.ts, called from main.ts
- getDailyAttempt|recordDailyAttempt in StorageManager.ts + TitleScreen.tsx + main.ts
- daily-stats in styles.css + TitleScreen.tsx
- clipboard.writeText|Share in GameOverScreen.tsx
- mode === 'daily' in main.ts, TitleScreen.tsx, GameOverScreen.tsx
- tsc --noEmit exits 0
- Bundle <= 250KB gzip
