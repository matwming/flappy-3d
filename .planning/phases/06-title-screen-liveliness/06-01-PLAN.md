Concrete changes to src/main.ts:

1. After `let prevState: string | undefined` add:
```typescript
let bobTime = 0
```

2. Before `loop.start()` add the bob step:
```typescript
loop.add({
  step: (dt: number) => {
    const s = actor.getSnapshot().value
    if (s !== 'title') {
      // Not title: reset bobTime, ensure mesh.y matches physics position (y=0 on title)
      if (s === 'playing') bobTime = 0  // only reset on play start; dying etc leave mesh alone
      return
    }
    if (prefersReducedMotion(storage)) {
      bird.mesh.position.y = bird.position.y  // flat, no bob
      return
    }
    bobTime += dt
    bird.mesh.position.y = bird.position.y + Math.sin(bobTime * Math.PI * 2) * 0.15
  },
})
```

Rationale for writing `bird.mesh.position.y` not `bird.position.y`:
- In title state PhysicsSystem gate prevents it running, so bird.position.y stays 0.
- When START fires, roundStarted hook calls `bird.position.set(0,0,0); bird.syncMesh()` which
  resets mesh.position.y to 0 cleanly before PhysicsSystem takes over.

3. In the actor.subscribe block, update the `title` branch to lower the music volume.
