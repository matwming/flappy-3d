```css
  --focus-inner: #ffd166;  /* yellow works for both palettes */

```

**Part B — Replace the existing `#ui-root button` rule:**

The existing rule (lines 26-43):
```css
#ui-root button {
  pointer-events: auto;
  min-width: 44px;
  min-height: 44px;
  padding: 12px 20px;
  border: 2px solid rgba(26, 26, 46, 0.25);
  background: rgba(255, 255, 255, 0.85);
  color: #1a1a2e;
  border-radius: 12px;
  font: inherit;
  font-weight: 600;
  cursor: pointer;
  backdrop-filter: blur(8px);
  transition: transform 150ms, background 150ms;
}
#ui-root button:hover { transform: translateY(-1px); background: rgba(255, 255, 255, 0.95); }
#ui-root button:focus-visible { outline: 2px solid #ffd166; outline-offset: 2px; }
#ui-root [role="switch"]:focus-visible { outline: 2px solid #ffd166; outline-offset: 2px; }
```

Replace with:
```css
#ui-root button {
  pointer-events: auto;
  min-width: 44px;
  min-height: 44px;
  padding: 12px 20px;
  border: none;
  background: linear-gradient(180deg, var(--btn-top) 0%, var(--btn-bottom) 100%);
  color: #fff;
  border-radius: 12px;
  font: inherit;
  font-weight: 600;
  cursor: pointer;
  box-shadow:
    inset 0 1px 0 rgba(255, 255, 255, 0.15),
    0 2px 4px rgba(0, 0, 0, 0.35);
  transition: transform 100ms ease-out, box-shadow 100ms ease-out;
}
@media (hover: hover) {
  #ui-root button:hover {
    transform: translateY(-1px);
    box-shadow:
      inset 0 1px 0 rgba(255, 255, 255, 0.15),
      0 4px 10px rgba(0, 0, 0, 0.4);
  }
}
#ui-root button:active {
  transform: translateY(0);
  box-shadow:
    inset 0 2px 5px rgba(0, 0, 0, 0.35),
    0 1px 2px rgba(0, 0, 0, 0.2);
}

/* ── Focus ring (BEAUTY-12) — replaces Phase 4 outline ── */
#ui-root button:focus-visible,
#ui-root [role="switch"]:focus-visible {
  outline: none;
  box-shadow:
    0 0 0 2px var(--focus-inner),
    0 0 0 4px var(--focus-outer),
    inset 0 1px 0 rgba(255, 255, 255, 0.15),
    0 2px 4px rgba(0, 0, 0, 0.35);
}
```

Notes on the focus rule:
- The third and fourth box-shadow layers preserve the button's resting depth so the focus ring adds ON TOP of it.
- Using `0 0 0 2px` (inner yellow) and `0 0 0 4px` (outer dark halo) gives 2-layer ring.
- `#ffd166` at 2px on the dark button gradients (contrast ratio: ~9:1 against `#2c3a64`; ~7:1 against `#4a5b8c`) is WCAG-AA compliant.
- The outer dark halo (`rgba(0,0,0,0.7)`) provides contrast when the button is displayed on a lighter background.
- The `install-cta` button has its own rule and its own `background` — the `:root` var gradient does NOT apply to it (it uses `background: rgba(...)` not `background: linear-gradient(...)`). The `.install-cta` rule has no `:focus-visible` override — it will inherit the new box-shadow focus rule since it's a button inside `#ui-root`. This is acceptable behavior (the yellow ring on the install CTA is fine).
