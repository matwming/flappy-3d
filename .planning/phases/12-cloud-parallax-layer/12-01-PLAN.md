# Phase 12, Plan 01: Cloud Parallax Layer

## Objective

Add 5 inline-SVG cloud sprite meshes that drift at 0.5× scroll speed between the sky shader and mountains, giving every mode a sense of atmospheric depth with zero network fetches.

## Context

- `src/entities/Background.ts` — pattern: scene meshes managed by an entity class
- `src/systems/ScrollSystem.ts` — state-gating reference (title + playing + dying active; paused + gameOver frozen)
- `src/main.ts` — loop.add registration, roundStarted hook
- `src/loop/GameLoop.ts` — `UpdatableSystem { step(dt: number): void }` contract
- `src/systems/Difficulty.ts` — `difficultyFrom(score).scrollSpeed`

## Tasks

### Task 1: Clouds entity with inline-SVG sprite + scroll behavior

**Type:** auto

Create `src/entities/Clouds.ts`:

- Named imports: `PlaneGeometry`, `MeshBasicMaterial`, `Mesh`, `TextureLoader`, `Scene` from 'three'
- Inline SVG cloud shape as utf-8 data URL (no network fetch)
- Shared `PlaneGeometry(2.5, 1.25)` and shared `MeshBasicMaterial({ map, transparent: true, depthWrite: false, opacity: 0.7 })`
- 5 cloud meshes pre-warmed, randomized x ∈ [-15, +15], y ∈ [1.5, 3.5], z = -7, scale ∈ [0.7, 1.3]
- `step(dt, scrollSpeed)`: each cloud x -= scrollSpeed * 0.5 * dt; when x < -20, respawn at +20 with new y/scale
- `reset()`: re-randomize all cloud positions (called on roundStarted)

**Done criteria:**
- `tsc --noEmit` exits 0
- Grep: `data:image/svg+xml` in Clouds.ts
- Grep: `transparent: true` and `depthWrite: false` in Clouds.ts

**Commit:** `feat(12-01): add Clouds entity with inline-SVG parallax sprites (ATMOS-01, ATMOS-02)`

### Task 2: Wire Clouds into main.ts loop + roundStarted reset

**Type:** auto

Update `src/main.ts`:

- Import `Clouds` from `./entities/Clouds`
- Instantiate `const clouds = new Clouds(scene)` after `const background = new Background(scene)`
- Add to loop: `loop.add({ step: (dt) => { const s = actor.getSnapshot().value; if (s === 'title' || s === 'playing' || s === 'dying') clouds.step(dt, s === 'title' ? 1.8 : difficultyFrom(actor.getSnapshot().context.score).scrollSpeed) } })`
- Add `clouds.reset()` to the roundStarted handler (alongside existing resets)
- Import `difficultyFrom` if not already present (it is not — add import)
