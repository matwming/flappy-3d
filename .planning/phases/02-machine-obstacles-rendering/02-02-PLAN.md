Generic pool with exactly this API:
```typescript
export class ObjectPool<T> {
  private free: T[]
  private active: T[]

  constructor(factory: () => T, size: number)

  acquire(): T | null
  // Returns an item from free list, moves it to active list.
  // Returns null ONLY if pool is exhausted (should never happen if sized right).
  // Does NOT call reset — caller (ObstacleSpawner) calls pair.reset() after acquire.

  release(item: T): void
  // Moves item from active list back to free list.

  forEachActive(cb: (item: T) => void): void
  // Iterates active items. Used by ScrollSystem, ScoreSystem, CollisionSystem.
}
```

TypeScript strict: use `readonly` for factory reference, proper generic constraint.
