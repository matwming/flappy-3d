Since the project may not have `canvas` installed, use this fallback strategy:

2. If canvas is not available, use the `node:fs` + base64-embedded minimal PNG approach: embed tiny valid 1×1 placeholder PNGs and scale them via a pure-JS PNG encoder stub — BUT this is too complex. Instead use a simpler approach:

**Actual implementation plan (no external deps):**
Write a script that creates PNG files using the `sharp` npm package if available, or uses raw Node.js Buffer to write a minimal valid PNG with the correct dimensions and a solid sky-blue (#7ec8e3) background with a centered orange (#ff7043) circle (bird silhouette).

**Simplest reliable approach — use raw PNG creation:**
Write a minimal PNG encoder in pure JS (no deps). A PNG file is: 8-byte signature + IHDR chunk + IDAT chunk (zlib-compressed scanlines) + IEND chunk. Use Node's built-in `zlib.deflateSync` for compression.

Implement `createSolidPNG(width, height, bgR, bgG, bgB, circR, circG, circB, cx, cy, radius)`:
- Fills background with bgRGB
- Draws a filled circle at (cx, cy) with given radius in circRGB
- Writes raw RGBA scanlines → deflateSync → wrap in PNG chunks

- `public/icons/icon-maskable-512.png` — 512×512 same as icon-512.png (maskable icons need 20% safe-zone padding — reduce circle radius to 140 and keep center)

```typescript
import { defineConfig } from 'vite'
import preact from '@preact/preset-vite'
import { visualizer } from 'rollup-plugin-visualizer'
import { VitePWA } from 'vite-plugin-pwa'

export default defineConfig({
  base: '/flappy-3d/',
  plugins: [
    preact(),
    VitePWA({
      registerType: 'autoUpdate',
      strategies: 'generateSW',
      workbox: {
        skipWaiting: true,
        clientsClaim: true,
        globPatterns: ['**/*.{js,css,html,png,svg,ico,mp3}'],
        runtimeCaching: [
          {
            urlPattern: /\.mp3$/i,
            handler: 'CacheFirst',
            options: {
              cacheName: 'audio-cache',
              expiration: { maxEntries: 10, maxAgeSeconds: 60 * 60 * 24 * 365 },
              cacheableResponse: { statuses: [0, 200] },
            },
          },
        ],
      },
      includeAssets: ['icons/icon-192.png', 'icons/icon-512.png', 'icons/icon-maskable-512.png'],
      manifest: {
        name: 'Flappy 3D',
        short_name: 'Flappy 3D',
        description: 'A polished 3D Flappy Bird PWA.',
        theme_color: '#7ec8e3',
        background_color: '#1a1a1a',
        display: 'standalone',
        orientation: 'portrait',
        start_url: '/flappy-3d/',
        scope: '/flappy-3d/',
        icons: [
          {
            src: 'icons/icon-192.png',
            sizes: '192x192',
            type: 'image/png',
          },
          {
            src: 'icons/icon-512.png',
            sizes: '512x512',
            type: 'image/png',
          },
          {
            src: 'icons/icon-maskable-512.png',
            sizes: '512x512',
            type: 'image/png',
            purpose: 'any maskable',
          },
        ],
      },
    }),
    visualizer({
      filename: 'dist/stats.html',
      open: false,
      gzipSize: true,
      brotliSize: false,
    }),
  ],
})
```
