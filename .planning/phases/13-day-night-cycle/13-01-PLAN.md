# Phase 13 Plan 01: Day/Night Cycle on Sky Shader

## Objective

Animate the existing sky ShaderMaterial uniforms `uTopColor` and `uBottomColor` between 4 keyframe color pairs over a 60-second looping cycle. Motion-gated via `prefersReducedMotion`. Cycle resets on `roundStarted` so each round starts at morning.

## Context

- `src/entities/Background.ts` ‚Äî sky shader lives here; uniforms `uTopColor`/`uBottomColor` already wired
- `src/constants.ts` ‚Äî extend with `SKY_KEYFRAMES` + `SKY_CYCLE_DURATION_S`
- `src/main.ts` ‚Äî loop + roundStarted hook
- `src/a11y/motion.ts` ‚Äî `prefersReducedMotion(storage)` helper

## Tasks

### Task 1: Add SKY_KEYFRAMES + extend Background.ts with cycleSky/resetSkyCycle

**Files:**
- `src/constants.ts` ‚Äî add `SKY_KEYFRAMES` (4 color pairs) and `SKY_CYCLE_DURATION_S = 60`
- `src/entities/Background.ts` ‚Äî add `private cycleElapsed = 0`, `cycleSky(dt, isReducedMotion)`, `resetSkyCycle()`

**Details:**
- `SKY_KEYFRAMES[0]` = morning (replaces current inline `SKY_TOP_COLOR`/`SKY_BOTTOM_COLOR` in `createSky()`)
- `cycleSky(dt, isReducedMotion)`: if reduced motion, return early; otherwise increment elapsed, compute idx/t, lerp uniforms
- `resetSkyCycle()`: reset elapsed to 0, snap uniforms to keyframe 0
- Remove now-unused module-level `SKY_TOP_COLOR` / `SKY_BOTTOM_COLOR` constants in Background.ts

**Commit:** `feat(13-01): add sky keyframes + Background.cycleSky() with prefersReducedMotion gate (ATMOS-03, ATMOS-04)`

### Task 2: Wire cycle into main.ts loop + roundStarted reset

**Files:**
- `src/main.ts` ‚Äî add loop step calling `background.cycleSky(dt, prefersReducedMotion(storage))`, add `background.resetSkyCycle()` in roundStarted handler

**Commit:** `feat(13-01): wire sky cycle in main.ts loop + roundStarted reset (ATMOS-03)`
